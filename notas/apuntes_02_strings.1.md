# Los Strings

* `len` longitud string

```python
    len(verse)
```

* `find` para buscar un string

```python
    index = verse.find("and",0,longitud)
```

* `rfind` para buscaro de forma inversa , desde el final al principio

```python
    index = verse.rfind("you",0,longitud)
```

* `count` ocurrencias en un string

```python
    verse.count("you")
```
